## ğŸš€ Getting Started

### Option 1: Direct Open
Simply open the `index.html` file in any modern web browser.

```bash
# Clone the repository
git clone https://github.com/yourusername/adhd-superpowers.git

# Navigate to the directory
cd adhd-superpowers

# Open in browser
open index.html
```

### Option 2: Live Server
Use VS Code's Live Server extension or any local development server for the best experience.

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

Then visit `http://localhost:8000` in your browser.
